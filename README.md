# Mini calculette web (Git Flow)

## Objectif
Mini-projet web : 2 champs numériques, choix d'opération, bouton *Calculer* et affichage du résultat.

## Fonctionnalités
- [x] Addition
- [x] Soustraction
- [x] Multiplication

## Fichiers
- index.html
- styles.css
- calculator.js
- README.md

## Lancer
Ouvrir `index.html` dans un navigateur.
